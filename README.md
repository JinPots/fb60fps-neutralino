# fb60fps-neutralino

## Neutralino / Lightweight version of facebook60fps-re-encoder

![Facebook 60FPS Re-Encoder Screenshot](https://cdn.discordapp.com/attachments/702488640525172787/1031514730227052594/unknown.png)
